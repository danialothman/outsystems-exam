  pip install gunicorn

KNOWN LIMITATIONS
================================================================================

1. Session-based only
   - Exam progress lost if browser closes
   - Best for single-sitting exams

2. No persistent database
   - All data stored in server memory
   - Not suitable for very large deployments

3. Single user at a time (development mode)
   - With proper deployment can handle multiple users
   - See DEPLOYMENT_GUIDE.md for scaling

4. Desktop-optimized
   - Mobile support is basic
   - Best experience on desktop/tablet

5. No built-in user authentication
   - Add authentication if deploying publicly
   - Currently honor-system
