INSTALLATION (2 minutes)
================================================================================

Option 1 - Using run.sh (Linux/macOS):
  1. cd outsystems-exam
  2. chmod +x run.sh
  3. ./run.sh
  4. Open http://localhost:5000

Option 2 - Manual (Any OS):
  1. cd outsystems-exam
  2. pip install -r requirements.txt
  3. python app.py
  4. Open http://localhost:5000

That's it! No database, no configuration needed.

TAKING THE EXAM
================================================================================

1. Timer starts automatically (120 minutes)
2. Click question numbers in sidebar to navigate
3. Select answers - they auto-save
4. Flag difficult questions with flag button
5. Use Previous/Next to navigate sequentially
6. Click "Submit Exam" when done (or at time limit)
7. Results show immediately with:
   - Final score (X/50)
   - Pass/Fail status
   - Category breakdown
   - Copy-to-clipboard export

AFTER THE EXAM
================================================================================

Copy all your answers:
1. Click "Copy All Answers to Clipboard" button
2. Paste into a text file or email
3. Share with others for grading
4. No correct answers are included (safe to share)

TROUBLESHOOTING
================================================================================

Problem: "Port 5000 already in use"
Solution: Edit app.py, change 'port=5000' to 'port=8080'

Problem: "Flask not found"
Solution: pip install flask

Problem: "Questions not loading"
Solution: Check Flask console for errors, refresh page

Problem: "Timer not syncing"
Solution: Refresh page (F5 or Cmd+R)

For more help, see DEPLOYMENT_GUIDE.md
